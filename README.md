# OpenMind

**CPT208 Human-Centric Computing**  
**Group Discussion Portfolio | Group B1-3**

OpenMind is a classroom discussion project designed to make participation more balanced, visible, and comfortable for both students and teachers. The portfolio documents our research, ideation, technical implementation, testing, evaluation, and AI usage reflections for the Group Discussion theme.

## Project Overview

The project focuses on Group Discussion participation. It aims to support quieter students, help teachers understand discussion patterns more clearly, and keep the interaction process structured without removing the human side of communication.

The system combines:

- a student-facing discussion flow,
- a teacher monitoring view,
- low-pressure participation support,
- and AI-assisted guidance for selected parts of the workflow.
